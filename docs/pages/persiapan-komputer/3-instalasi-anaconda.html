3. Instalasi Anaconda

Sebenarnya, ada banyak python package and environment manager seperti virtualenv, Poetry, dan uv. Di sini penulis memilih Anaconda karena penulis merasa Anaconda adalah environment manager yang cukup beginner friendly.

Pertama, unduh installer dari Anaconda. Tautan untuk installer nya dapat ditemukan di Official website Anaconda

xxxxxxxxxx
wget https://repo.anaconda.com/archive/Anaconda3-2025.12-2-Linux-x86_64.sh

Lakukan instalasi dengan

xxxxxxxxxx
bash Anaconda3-2025.12-2-Linux-x86_64.sh

Lakukan proses instalasi sampai bertemu pilihan ini

xxxxxxxxxx
Proceed with initialization? [yes|no]
[no] >>>

Pilih yes agar tiap log in, conda akan otomatis diaktifkan dan dimasukkan ke dalam PATH. Anaconda terbukti berhasil di-install dengan munculnya tanda (base) di awal tiap terminal line seperti ini

xxxxxxxxxx
(base) fahadh@localhost:~ $
