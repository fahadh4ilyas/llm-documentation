1. Instalasi Nvidia Driver & Cuda Toolkit

Package yang pertama adalah Nvidia Driver. Package ini bertujuan agar komputer dapat mengakses GPU yang telah dipasang pada komputer yang nanti akan digunakan dalam proses load, inference, dan fine tune LLM. Selalu pastikan install driver yang terbaru dari nvidia. Di luaran, orang-orang bilang versi driver harus menyesuaikan dengan versi cuda toolkit. Padahal, setiap driver nvidia terbaru selalu back compatible dengan versi cuda toolkit lama. Karena itu, selalu pilih versi Nvidia Driver terbaru yang tersedia pada OS komputer.

Nvidia Driver

Untuk meng-install nvidia driver, install terlebih dahulu cuda-keyring package terbaru. Isi <distro> and <arch> sesuai spesifikasi komputer kalian

xxxxxxxxxx
wget https://developer.download.nvidia.com/compute/cuda/repos/<distro>/<arch>/cuda-keyring_1.1-1_all.deb;
sudo dpkg -i cuda-keyring_1.1-1_all.deb;
sudo apt update;

Sebagai contoh, untuk Ubuntu 22.04 dengan arsitektur amd64, <distro> nya adalah ubuntu2204 and <arch> nya adalah x86_64. Kemudian, install nvidia driver yang dibutuhkan. Terakhir pada tanggal 8 Juli 2026, versi terbaru dalam OS Ubuntu 22.04 adalah versi 595

xxxxxxxxxx
sudo apt install nvidia-driver-595 -y

Kalian juga bisa memilih versi open kernel dari Nvidia Driver nvidia-driver-595-open. Secara performa, tidak ada perbedaan dari keduanya. Tetapi, versi open kernel lebih direkomendasikan untuk arsitektur GPU terbaru (Turing, Ampere, Ada Lovelace, Blackwell, Hopper, dll).

Untuk memeriksa versi Nvidia Driver yang telah di-install, dapat dilakukan dengan perintah berikut

xxxxxxxxxx
cat /proc/driver/nvidia/version

Cuda Toolkit

Selanjutnya adalah meng-install Cuda Toolkit. Untuk ini, kalian bebas meng-install versi berapapun. Akan disarankan untuk meng-install beberapa versi dari cuda berdasarkan kebutuhan kalian. Penulis biasa meng-install satu versi major 12 dan satu versi major 13. Jadi, penulis akan meng-install Cuda Toolkit versi 12.9 dan versi 13.2.

xxxxxxxxxx
sudo apt install cuda-toolkit-12-9 -y;
sudo apt install cuda-toolkit-13-2 -y;

Untuk memeriksa apakah Cuda Toolkit sudah ter-install pada komputer kalian, bisa diperiksa isi dari direktori /usr/local. Akan ada beberapa direktori seperti berikut:

  • cuda

  • cuda-12 (jika meng-install Cuda Toolkit versi major 12)

  • cuda-13 (jika meng-install Cuda Toolkit versi major 13)

  • cuda-12.9

  • cuda-13.2

Untuk memilih versi Cuda Toolkit yang ingin digunakan, kalian bisa menambahkan /usr/local/cuda-<version>/bin ke dalam environment variable PATH dan /usr/local/cuda-<version>/lib64 ke dalam LD_LIBRARY_PATH. Penulis biasa menulisnya ke dalam ~/.bashrc seperti berikut

xxxxxxxxxx
export PATH=/usr/local/cuda-13.2/bin${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-13.2/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

Setelah menambahkan ke dalam environment variable PATH dan LD_LIBRARY_PATH, untuk memeriksa versi Cuda Toolkit yang sudah di-install dapat dilakukan dengan perintah berikut

xxxxxxxxxx
nvcc --version
