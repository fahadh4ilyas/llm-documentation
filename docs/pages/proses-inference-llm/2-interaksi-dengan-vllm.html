2. Interaksi Dengan vLLM

Setelah berhasil deploy LLM menggunakan vLLM, kita bisa berinteraksi dengan LLM menggunakan openAI API. Kalian bisa berinteraksi dengan API tersebut melalui Swagger. Misal vLLM yang di-deploy menggunakan port 8000. Kalian bisa akses ke http://localhost:8000/docs untuk melihat Swagger dari API yang telah di-deploy. Berikut adalah contoh tampilannya

Kalian bisa coba dengan memilih collapsible /v1/chat/completions dan memasukkan contoh JSON ini ke dalam Request body

xxxxxxxxxx
{
    "messages": [
        {
            "role": "user",
            "content": "This is a test. Write me a short poem!"
        }
    ],
    "model": "Qwen3.5-9B"
}

Setelah kalian tekan tombol Execute, LLM akan memroses input dan akan mengeluarkan jawaban seperti contoh berikut

Berikut adalah parameter-parameter yang ada dalam body dengan endpoint /v1/chat/completions

Untuk pembuatan tools, biasanya tools berisikan daftar dari tool yang merupakan JSON schema dari function yang kalian buat. Berikut adalah format dari tool

x
{
    "type": "function",
    "function": {
        "name": <function_name: str>,
        "description": <function_description: str>,
        "parameters": {
            "type": "object",
            "properties": <properties: object>,
            "required": <required_list: str[]>
        }
    }
}

Sebagai contoh, jika kalian punya tools berupa tool untuk menjumlahkan dua angka dan tool untuk mendapatkan waktu saat ini. Maka, tools yang kalian tuliskan seperti berikut

xxxxxxxxxx
[
    {
        "type": "function",
        "function": {
            "name": "add",
            "description": "Add two numbers together and return the sum",
            "parameters": {
                "type": "object",
                "properties": {
                    "a": {"type": "number", "description": "The first number"},
                    "b": {"type": "number", "description": "The second number"},
                },
                "required": ["a", "b"]
            }
        }
    },
    {
        "type": "function",
        "function": {
            "name": "get_current_time",
            "description": "Get the current date and time",
            "parameters": {
                "type": "object",
                "properties": {}
            }
        }
    }
]  

Untuk parameter-parameter lainnya yang bisa digunakan di vLLM, dapat kalian lihat melalui tautan berikut.
