Proses Load LLM

Salah satu website tempat orang-orang menyebar dan menyimpan model secara online adalah huggingface. Kita bisa menggunakan website tersebut untuk mengunduh model LLM yang kita butuhkan. Sangat direkomendasikan untuk membuat akun huggingface. Untuk membuat akun huggingface, silakan klik tautan berikut.

Setelah membuat akun, disarankan untuk membuat token akses. Tujuannya adalah agar saat mengunduh model LLM, dapat kecepatan unduh yang optimal dan tidak dibatasi oleh hugging face. Untuk membuat token akses, silakan klik tautan berikut. Berikut adalah tampilan dari token akses

Pertama, klik + Create new token untuk membuat token baru. Berikut adalah tampilan awal saat membuat token baru.

Untuk mempermudah, klik tab Read. Kemudian masukkan nama token terserah kalian. Seperti ini contoh tampilannya

Setelah klik tombol Create token, akan muncul pop-up token yang telah dibuat. Langsung copy token tersebut karena tidak akan bisa diakses lagi. Berikut adalah contoh tampilannya

Simpan token tersebut. JANGAN DISEBARKAN KE ORANG LAIN!!!

Setelah mendapatkan token, tahap selanjutnya adalah mencari model yang ingin diunduh. Penulis akan mengambil contoh model Qwen3.5-9B yang bisa dilihat di tautan berikut. Yang kalian perlu ambil adalah nama dari model tersebut. Untuk Qwen3.5-9B adalah Qwen/Qwen3.5-9B.

Setelah menentukan model apa yang ingin diunduh, lakukan proses install package huggingface-hub. Buka terminal sampai muncul tampilan seperti ini

xxxxxxxxxx
(base) fahadh@localhost:~ $

Setelah dipastikan environment python yang kalian masuki adalahbase, lakukan instalasi package huggingface-hub dengan perintah berikut

xxxxxxxxxx
pip install huggingface-hub

Setelah proses instalasi selesai, kalian bisa periksa apakah package sudah ter-install dengan perintah berikut

xxxxxxxxxx
hf --help

Jika keluarannya adalah bantuan untuk perintah hf, artinya package huggingface-hub telah berhasil di-install di environment base. Tahap selanjutnya adalah proses mengunduh model.

Kalian bebas ingin menaruh model-model LLM kalian di mana saja. Penulis biasanya meletakkan seluruh model AI yang diunduh di direktori ~/models. Buat dulu direktorinya dengan perintah

xxxxxxxxxx
mkdir -p ~/models

Kemudian, untuk mengunduh model Qwen3.5-9B, bisa dilakukan dengan perintah

xxxxxxxxxx
hf download --local-dir ~/models/Qwen3.5-9B --token <token_access> Qwen/Qwen3.5-9B

Parameter --local-dir digunakan untuk menentukan di mana model akan diunduh dan parameter --token digunakan untuk token akses yang telah kalian dapatkan dari akun huggingface kalian. Setelah proses unduh selesai, bisa kalian lihat isi direktori hasil unduhan kalian

xxxxxxxxxx
ls ~/models/Qwen3.5-9B
